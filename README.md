# Python WebView Desktop App Demo

This Python example demonstrates the core concept behind lightweight desktop application frameworks like Verve, Tauri, and Wails. It uses the `pywebview` library to embed the operating system's native WebView, displaying HTML/CSS/JavaScript content without bundling a full web browser. This approach significantly reduces resource consumption and application size compared to Electron-based applications.

## Language

`python`

## How to Run

1. Install pywebview: `pip install pywebview`
2. Run the script: `python main.py`
